## 5분 버전
1. 배경: 기존 memorability 연구는 점수 예측 중심
2. 문제: 사용자가 실제로 바꿀 수 있는 피드백 부재
3. 제안: MemFeed + MemCoach(steering) + MemBench
4. 메시지: 측정에서 개선으로 패러다임 전환

## 10분 버전
1. 문제 정의와 사용자 시나리오
2. Teacher-Student contrastive generation
3. Steering vector 추출과 inference 적용
4. 벤치마크 구성과 결과 해석
5. 한계: 개인차, 문화차, 실제 촬영 제약
